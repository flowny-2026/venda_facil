📝 Como Adicionar as Imagens

  1. Crie ou baixe suas imagens usando as ferramentas sugeridas acima
  2. Otimize as imagens usando TinyPNG para reduzir o tamanho
  3. Salve os arquivos na pasta assets/images/ com os nomes exatos indicados
  4. Atualize o index.html substituindo os emojis pelas tags <img>
  5. Teste localmente abrindo o index.html no navegador
  6. Faça commit e push para o GitHub

Exemplo de código para usar as imagens:

<!-- Logo no header -->
<img src="assets/images/logo.png" alt="VendaFácil" style="height: 40px;">

<!-- Hero mockup -->
<img src="assets/images/hero-mockup.png" alt="Sistema VendaFácil" style="max-width: 100%;">

<!-- Favicon no head -->
<link rel="icon" type="image/x-icon" href="assets/images/favicon.ico">
