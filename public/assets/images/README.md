# 📁 Pasta de Imagens - VendaFácil

## 🎨 Imagens Necessárias

### **Logo**
- `logo.png` - Logo principal do VendaFácil (fundo transparente)
- `logo-white.png` - Logo branco para fundos escuros
- Tamanho recomendado: 200x60px ou similar
- Formato: PNG com transparência

### **Hero Section**
- `hero-mockup.png` - Mockup do sistema (notebook + celular)
- Tamanho recomendado: 1200x800px
- Formato: PNG ou JPG

### **Produto/Sistema**
- `dashboard-screenshot.png` - Screenshot do dashboard
- `pdv-screenshot.png` - Screenshot do PDV
- `mobile-app.png` - Mockup do app mobile
- Tamanho recomendado: 800x600px
- Formato: PNG ou JPG

### **Ícones** (opcional)
- Você pode usar emojis ou ícones SVG
- Ou adicionar ícones personalizados aqui

### **Outros**
- `favicon.ico` - Ícone do site (16x16px, 32x32px)
- `og-image.png` - Imagem para compartilhamento social (1200x630px)

## 📝 Como Usar

Após adicionar as imagens nesta pasta, atualize o `index.html`:

```html
<!-- Logo no header -->
<img src="assets/images/logo.png" alt="VendaFácil" class="logo-img">

<!-- Hero mockup -->
<img src="assets/images/hero-mockup.png" alt="Sistema VendaFácil">

<!-- Screenshots -->
<img src="assets/images/dashboard-screenshot.png" alt="Dashboard">
```

## 🎯 Dicas

1. **Otimize as imagens** antes de adicionar (use TinyPNG ou similar)
2. **Use WebP** quando possível para melhor performance
3. **Adicione alt text** descritivo em todas as imagens
4. **Mantenha tamanhos consistentes** para melhor carregamento
